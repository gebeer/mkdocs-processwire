Add a breadcrumb
----------------

### Example

    $this->breadcrumb("../", "Widgets"); 

### Usage

    $process->breadcrumb(string $href, string $label);

### Arguments

Name

Type(s)

Description

`href`

string

URL of breadcrumb

`label`

string

Label for breadcrumb

### Return value

`$this`

Object instance it was called from (method supports fluent interface).
