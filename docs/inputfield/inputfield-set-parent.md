Set the parent (InputfieldWrapper) of this Inputfield.
------------------------------------------------------

### Usage

    $inputfield->setParent(InputfieldWrapper $parent);

### Arguments

Name

Type(s)

Description

`parent`

InputfieldWrapper

### Return value

`$this`

Object instance it was called from (method supports fluent interface).
