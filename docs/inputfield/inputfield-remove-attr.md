Remove an attribute
-------------------

### Usage

    $inputfield->removeAttr(string $key);

### Arguments

Name

Type(s)

Description

`key`

string

Name of attribute to remove.

### Return value

`$this`

Object instance it was called from (method supports fluent interface).
