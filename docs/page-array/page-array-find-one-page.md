Same as find() or findOne() methods, but always returns a Page (whether Page or NullPage)
-----------------------------------------------------------------------------------------

Available since version **3.0.162**.

### Usage

    $page = $pageArray->findOnePage(string $selector);

### Arguments

Name

Type(s)

Description

`selector`

string

### Return value

[`Page`](/api/ref/page/) [`NullPage`](/api/ref/null-page/)
