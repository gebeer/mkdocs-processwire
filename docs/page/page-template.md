Get or set template
-------------------

Available since version **3.0.181**.

### Usage

    // basic usage
    $template = $page->template();
    
    // usage with all arguments
    $template = $page->template($template = null);

### Arguments

Name

Type(s)

Description

`template` (optional)

null, Template, string, int

### Return value

[`Template`](/api/ref/template/) `null`
